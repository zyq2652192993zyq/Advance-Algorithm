> # 一本通-1011：甲流疫情死亡率

### 【题目描述】

甲流并不可怕，在中国，它的死亡率并不是很高。请根据截止2009年12月22日各省报告的甲流确诊数和死亡数，计算甲流在各省的死亡率。

### 【输入】

输入仅一行，有两个整数，第一个为确诊数，第二个为死亡数。

### 【输出】

输出仅一行，甲流死亡率，以百分数形式输出，精确到小数点后3位。

### 【输入样例】

```
10433 60
```

### 【输出样例】

```
0.575%
```

-----

```c
#include <stdlib.h>
#include <stdio.h>

int main()
{
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%.3f%%\n", b * 1.0 / a * 100);

	return 0;
}
```

输出百分数两个百分号。
