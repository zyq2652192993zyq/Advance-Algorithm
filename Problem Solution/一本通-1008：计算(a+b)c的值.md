> # 一本通-1008：计算(a+b)/c的值

### 【题目描述】

给定33个整数a、b、ca、b、c，计算表达式a+bca+bc的值。

### 【输入】

输入仅一行，包括三个整数a、b、ca、b、c, 数与数之间以一个空格分开。(－10,000<a,b,c<10,000－10,000<a,b,c<10,000, cc不等于00)

### 【输出】

输出一行，即表达式的值。

### 【输入样例】

```
1 1 3
```

### 【输出样例】

```
0
```

----

```c
#include <stdlib.h>
#include <stdio.h>

int main()
{
	int a, b, c;
	scanf("%d %d %d", &a, &b, &c);
	printf("%d\n", (a + b) / c);

	return 0;
}
```
