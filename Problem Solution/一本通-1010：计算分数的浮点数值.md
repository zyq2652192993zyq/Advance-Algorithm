> # 一本通-1010：计算分数的浮点数值

### 【题目描述】

两个整数a和b分别作为分子和分母，既分数a/b，求它的浮点数值(双精度浮点数，保留小数点后99位)。

### 【输入】

输入仅一行，包括两个整数a和b。

### 【输出】

输出也仅一行，分数a/b的浮点数值（双精度浮点数，保留小数点后99位）。

### 【输入样例】

```
5 7
```

### 【输出样例】

```
0.714285714
```

----

```c
#include <stdlib.h>
#include <stdio.h>

int main()
{
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%.9f\n", a * 1.0 / b);

	return 0;
}
```
