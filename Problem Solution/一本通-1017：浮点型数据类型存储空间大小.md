> # 一本通-1017：浮点型数据类型存储空间大小

### 【题目描述】

分别定义float，double类型的变量各一个，并依次输出它们的存储空间大小(单位：字节)。

### 【输入】

(无)

### 【输出】

一行，两个整数，分别是两个变量的存储空间大小，用一个空格隔开。

### 【输入样例】

(无)

### 【输出样例】

(无)

----

```c
#include <stdlib.h>
#include <stdio.h>

int main()
{
	printf("%d %d\n", (int)sizeof(float), (int)sizeof(double));

	return 0;
}
```
