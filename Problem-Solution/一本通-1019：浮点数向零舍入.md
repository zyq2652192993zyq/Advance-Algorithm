> # 一本通-1019：浮点数向零舍入

### 【题目描述】

输入一个单精度浮点数，将其向零舍入到整数。说明：向零舍入的含义是，正数向下舍入，负数向上舍入。提示：可以使用强制类型转换来实现。

### 【输入】

一个单精度浮点数。

### 【输出】

一个整数，即向零舍入到整数的结果。

### 【输入样例】

2.3

### 【输出样例】

2

-----

```c
#include <stdlib.h>
#include <stdio.h>
#include <stdbool.h>

int main()
{
	double num;
	scanf("%lf", &num);
	printf("%d\n", (int)num);

	return 0;
}
```

如果向上取整要用`ceil`，向下取整是`floor`，GCC编译的时候编译选项要加上`-lm`，否则报错。
